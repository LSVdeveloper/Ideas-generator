Генератор идей для досуга.

На главном экране: Настройки и Выход.
Список:
1.Досуг для одного человека;
2.Досуг для компании;
3.Досуг с ребенком(несколько пунктов с различным возрастом детей);
4.И что же почитать?

При нажатии на пункт списка появляется экран с кнопкой: "Старт";
При нажатии на кнопку на экран выводится текст с идеей, текст кнопки меняется на "Другая идея";

Информация хранится в базе данных по одной таблице на каждый пункт списка.
